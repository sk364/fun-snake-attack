Fun-snake-attack

A two-player extended version of Snake game developed in Python using Pygame.

Download this project as a .zip file Download this project as a tar.gz file

About

Snake Game is about a snake trying to eat food in an area without bumping into itself or the game is over. This extended version of snake consists of a two-player version where there are two snakes fighting for a single food block.

Installation

The game is developed using Python 2.7 and pygame module in it. Install Python 2.7 and finally install Pygame to complete your installation.

Run the Game

Run main.py using following command - python2.7 main.py to run the game.
